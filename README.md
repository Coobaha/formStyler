jQuery Form Styler
==================

jQuery-плагин для стилизации элементов html-форм:

* `<input type="checkbox">`
* `<input type="radio">`
* `<input type="file">`
* `<select>`

[Домашняя страница плагина] (http://dimox.name/jquery-form-styler/)
[Страница с примерами] (http://dimox.github.com/formStyler/)
